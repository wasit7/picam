To take open the camera
1. make sure the flash drive with label 'algae' 
   is attach in the USB port.
2. open terminal
3. use a command
   ~/github/picam/run.sh
4. when the camera is ready, the live image 
   from carema will display on the screen.
5. press 'Enter' to save image or
   press 'x then Enter' to exit the program


The recorded file are in '/media/algae/'.
